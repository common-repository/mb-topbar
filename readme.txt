This is a theme demo bar that allow developers / designers to showcase their designs / websites.

== Description ==

This plugin adds demo bar on top of the page. It is suitable for web designers and web developers. If you are theme designer or developer and if you would like to showcase your designs this plugin could help.

= Key Features =

+ **Responsive Test Tool.** Change website width on the fly., Visible only on desktop.
+ **Social Share Button.** Share to Facebook, Twitter and Linkedin.
+ **Logo Upload.** Change the logo.
+ **Background Color.** Choose background color of the showcase item topbar.
+ **Showcase Name.** Select item name.
+ **Slug.** Slug for the address bar.

**Contribute:**

This plugin is open source software, and you may actively contribute on [Github](https://github.com/Tihi321//mb-topbar "MB Topbar").


== Installation ==

1. Upload the plugin files to the `/wp-content/mb-topbar\' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress
3. Use the MB Topbar screen to configure the plugin

== Frequently Asked Questions ==

Q: What are the Wordpress requirements ?
A: Wordpress version 5.0 or above is required as React is enqueued from Wordpress.
Q: How to add a showcase ?
A: Under MB Topbar->Showcase choose tad Add Showcase .
